# Sonic 2D Game - C + Raylib

Projeto acadêmico de um jogo 2D inspirado em Sonic, desenvolvido em linguagem C utilizando recursos da biblioteca Raylib e sprites baseados nos jogos clássicos da franquia.

O objetivo do projeto foi praticar lógica de programação, organização de código, manipulação de sprites, movimentação de personagens, colisões, inimigos e construção de fases em um ambiente 2D.

## Sobre o projeto

Este jogo foi desenvolvido como forma de aplicar conceitos de programação em C em um projeto prático e visual. A ideia foi recriar mecânicas inspiradas em jogos clássicos de plataforma, como movimentação lateral, pulo, colisão com o cenário, inimigos e elementos interativos.

O projeto utiliza a Raylib para facilitar a parte gráfica, renderização, entrada de teclado e controle da janela do jogo.

## Tecnologias utilizadas

- Linguagem C
- Raylib
- Makefile
- Sprites 2D
- Programação estruturada
- Lógica de colisão
- Manipulação de animações

## Funcionalidades

- Movimentação lateral do personagem
- Sistema de pulo
- Colisão com plataformas e cenário
- Inimigos no mapa
- Sprites e animações
- Sistema de fases
- Elementos inspirados em jogos clássicos de plataforma
- Renderização 2D com Raylib

## Objetivos de aprendizado

Durante o desenvolvimento, foram trabalhados conceitos como:

- Estruturação de um projeto em C
- Separação de responsabilidades no código
- Uso de biblioteca gráfica
- Controle de entrada pelo teclado
- Atualização de estados do jogo
- Detecção de colisão
- Controle de sprites e animações
- Organização de arquivos com Makefile

## Como executar

Para rodar o projeto, é necessário ter a Raylib instalada e configurada no ambiente.

Depois, basta abrir o projeto no VS Code e executar o comando abaixo no terminal:

```bash
make
